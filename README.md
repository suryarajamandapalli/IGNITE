- **URL**: `index.html`
- **Features**: Real-time stats, charts, activity feed
- **Parameters**: None
- **Auto-refresh**: Every 30 seconds

### **Jobs View** (Tab Navigation)
- **Access**: Click "Jobs" tab or press `2`
- **Features**: Job listing, search, filtering, pagination
- **Filters**: Status (all, running, queued, completed, failed)

### **Analytics View** (Tab Navigation)
- **Access**: Click "Analytics" tab or press `3`
- **Features**: Advanced charts, utilization metrics, trends
- **Metrics**: Success rates, wait times, machine performance

### **Machines View** (Tab Navigation)
- **Access**: Click "Machines" tab or press `4`
- **Features**: Backend status, specifications, queue information
- **Data**: Live status, qubit counts, fidelity, uptime
